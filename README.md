# M3u8Downloader_H.Plugins
本项目是M3u8Downloader_H项目得插件项目，此项目得目的是为了能够与主程序分离，插件更新或者升级不会影响到主程序，也就是说插件可以完成动态下载，或者动态更新。

# 插件列表
 - 以下表格中得站点将支持直接使用视频页面得地址来进行下载
 - 插件还在逐步增加中
